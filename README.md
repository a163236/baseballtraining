# 野球部監督（Baseball Training）

高校野球の監督として、部員を育成し、地区大会から甲子園優勝を目指す育成シミュレーションです。  
Flutter で Android / iOS / macOS 向けに動作します。

## 必要な環境

1. [Flutter SDK](https://docs.flutter.dev/get-started/install)（3.2 以上推奨）
2. 各プラットフォームの開発環境（Android Studio、Xcode など）

インストール後、ターミナルで次を確認してください。

```bash
flutter doctor
```

## 初回セットアップ

このフォルダにはゲーム本体（`lib/`）だけが含まれています。初回はプラットフォーム用ファイルを生成します。

```bash
cd c:\Users\nagak\games\baseballtraining
flutter create . --project-name baseball_training
flutter pub get
```

`flutter create .` は既存の `lib/main.dart` を上書きしないようマージされます。上書きされた場合は、リポジトリの `lib/` を再度使ってください。

## 起動方法

### Android（実機またはエミュレータ）

```bash
flutter devices
flutter run
```

### iOS（Mac + Xcode が必要）

```bash
flutter run -d ios
```

### macOS デスクトップ

```bash
flutter config --enable-macos-desktop
flutter run -d macos
```

## 遊び方

1. **タイトル画面**で高校名を入力し、「新しく始める」または「続きから」を選びます。進行状況は自動保存されます。
2. **ホーム画面**で現在の年・週、チーム戦力、大会進捗を確認します。
3. **毎週の練習**（週に1回）  
   「今週の練習を決める」から、球速・変化球・ミート・パワー・走力・守備力のどれを重点的に伸ばすか選びます。  
   選ばなかった能力も、全員わずかに成長します。
4. **翌週へ進む**で時間を進めます。練習をスキップして進めることもできます。
5. **試合**  
   夏の特定の週に公式戦が入ります。「試合結果を見る」で勝敗とスコアだけが表示されます（プレイ中のシミュレーションはありません）。  
   進行: 地区大会 → 県大会 → 甲子園 → 決勝（優勝）
6. **スカウト**（年1回程度）  
   新年度や冬の週に、1年生候補3人から1人をカウトできます。ロースター上限は18人です。
7. **部員一覧**（画面上部のアイコン）で各選手の6能力と総合値を確認できます。

### 選手の能力

| 能力 | 説明 |
|------|------|
| 球速 | 投手の速球 |
| 変化球 | 投手の変化球 |
| ミート | 打撃のミート |
| パワー | 長打力 |
| 走力 | 走塁 |
| 守備力 | 守備・送球 |

初期値は各能力 25〜50 のランダムです。3年生は卒業し、スカウトで補強します。

## トラブルシューティング

- `flutter` コマンドが見つからない → Flutter をインストールし、PATH に `flutter\bin` を追加してターミナルを開き直してください。
- `flutter create` 前に `flutter run` するとエラーになります → 上記「初回セットアップ」を実行してください。
